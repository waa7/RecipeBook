Sticky Sticky component positions other components as fixed so that these components stay in window viewport during scrolling. In this example, menubar component made sticky so it is always visible during page scroll.

Usage

Element to be made sticky is defined using target attribute. 
<div id="sticky-target"></div>

<p-sticky target="sticky-target"></p-sticky>

Attributes

Name Type Default Description
id string null Identifier of the element.
target string null Identifier of the target element.
renderdekay integer 0 Time in milliseconds to delay the initialiation.
