Name Type Default Description
activeIndex Integer 0 Index of the active tab.
orientation String top Orientation of tab headers.
Name Parameters Description
change ui.index: index of selected tab. Fired when an inactive tab is selected.
close ui.index: index of selected tab. Fired when a tab is closed.

Example

$('#default').puitabview({
    change: function(event, ui) {
        //...
    }
});
Name Parameters Description
select index: Index of the tab to select. Activates the tab with given index.
remove index: Index of the tab to remove. Removes the tab with given index.
getLength - Returns length of tabs.
getActiveIndex - Returns index of selected tab.
disable index: Index of the tab to disable. Disable selected tab.
enable index: Index of the tab to enable. Enable selected tab.
option name: Name of the option Returns the value of the option.
option name: Name of the option, value: Value of the option Set the value of the option.

Example

$('#default').puitabview('remove', 1);
                                
$(function() {       
    $('#default').puitabview();

    $('#dynamic').puitabview({
        change: function(event, ui) {

            $.ajax({
                type: "GET",
                url: './tabcontent',
                data: {tabindex: ui.index},
                dataType: "html",
                context: this,
                success: function(data) {
                    $(this).children('div.ui-tabview-panels').children().eq(ui.index).html(data);
                }
            });
        }
    });


    $('#effect').puitabview({
        effect: {
            name: 'fade'
            ,duration: 'normal'
        }
    });

    $('#closable').puitabview();

    $('#left').puitabview({
        orientation: 'left'
    });
});

Usage

Tabview element consists of one or more p-tab elements. Header of the tab is defined using title attribute of tab. 
<p-tabview>
    <p-tab title="Title for tab 1">
        Tab Content 1
    </p-tab>
    <p-tab title="Title for tab 2">
        Tab Content 2
    </p-tab>
</p-tabview>
