Usage

Slider is defined with p-slider element. 
<p-slider></p-slider>

Attributes

Name Type Default Description
animate boolean false Whether to slide the handle smoothly when the user clicks on the slider track.
disabled bollean false Disables the slider if set to true.
max number 100 Maximum value for slider.
min number 0 Mininum value for slider.
orientation string horizontal Determines whether the slider handles move horizontally (min on left, max on right) or vertically (min on bottom, max on top). Possible values: "horizontal", "vertical".
range boolean false Whether the slider represents a range.
step number 1 Determines the size or amount of each interval or step the slider takes between the min and max. The full specified value range of the slider (max - min) should be evenly divisible by the step.
value number 0 Determines the value of the slider, if there's only one handle. If there is more than one handle, determines the value of the first handle. It should be a comma separated value if range is enabled.
style string null Inline style of the element.
styleclass string null Style class of the element.
onchange string null Name of the function to invoke when slider value changes.
onslide string null Name of the function to invoke on mousemove during slide.
onstart string null Name of the function to invoke before slide begins.
onstop string null Name of the function to invoke on after slide ends.

Methods

Name Parameters Description
enable() - Enables the element.
disable() - Disables the element.
destroy() - Destroy the selected element.
setValue(val) val: A number or an array of numbers for range mode Updates the slider with given value(s).
getValue() - Returns the current value of slider. Value is an array of numbers in range mode. 
<script type="text/javascript">
    slideChange = function(events,value) {
        $('#slideChange').val(value.value)
    };

    $(function () {
        $('#slideChange').puiinputtext();
    });
</script>
