<script type="text/javascript">
        $(function() {
            $('#rating').puirating();
                
            $('#default').puirating();
                
            $('#callback').puirating({rate: function(event, value) { 
                    alert(value);
                }});
                
            $('#nocancel').puirating({cancel:false});
                
            $('#readonly').puirating({readonly:true, cancel:false});
                                         
        });
    </script>

Usage

Rating is used as an input component. 
<p-rating stars="10" name="rating-param"></p-rating>

Attributes

Name Type Default Description
id string null Identifier of the element.
stars integer 5 Number of stars.
cancel boolean true Defines if user can cancel rating.
name string null Name of the request parameter to be used in a form submission.
onrate function null Name of the function to invoke when rating changes.
oncancel function null Name of the function to invoke when rating is cleared by cancel option.

Methods

Name Parameters Description
cancel - Clears selection.
getValue - Returns current value.
setValue value: Value to update. Updates current value.
enable - Enable the element.
disable - Disables the element.
