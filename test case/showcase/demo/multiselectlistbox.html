MultiSelectListBox MultiSelectListbox is used to select an item from a collection of listboxes that are in parent-child relationship.

Name Type Default Description
caption string null Header of root listbox.
choices array null Choices for the list.
effect string slide Effect to use when showing subgroups.
name string null Name of the request parameter map key to be used when enclosing form is submitted.
value string null Value to show as preselected.

No Events.

Name Parameters Description
disable - Disables the list container.
getValue - Returns the value of selected option.
