Name Type Default Description
toggleable Boolean false Defines if content of fieldset can be expanded and collapsed.
toggleDuration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
collapsed Boolean false Displays the fieldset as collapsed initially.

beforeToggle event: puifieldsetbeforetoggle event
collapsed: Collapsed state as a boolean 		Fired before toggling.

Example

$('#default').puifieldset({

    }
});
Name Parameters Description
toggle - Toggles content.
option name: Name of the option Returns the value of the option.
option name: Name of the option, value: Value of the option Set the value of the option.

Usage

Fieldset is a container element having a legend and content as children.

<p-fieldset toggleable>
    <legend>PrimeElement</legend>
    Content
</p-fieldset>

Attributes

Name Type Default Description
id string null Identifier of the element.
toggleable Boolean false Defines if content of fieldset can be expanded and collapsed.
toggleduration Number/String normal Defines the duration of the toggle effect, valid values are "slow", "normal", "fast" or a number in milliseconds.
collapsed Boolean false Displays the fieldset as collapsed initially.
onbeforetoggle function event: puifieldsetbeforetoggle event
collapsed: Collapsed state as a boolean 		Name of the function to invoke before fieldset gets toggled.
onaftertoggle function event: puifieldsetaftertoggle event
collapsed: Collapsed state as a boolean 		Name of the function to invoke after fieldset gets toggled.

Methods

Name Parameters Description
toggle() - Toggles the content.
